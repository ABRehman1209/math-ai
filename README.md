# SolvIt - Math AI Solver 🎓✨

A powerful, modern Math AI that handles algebra, calculus, and equation solving with beautiful LaTeX rendering.

- **Smart Solving**: Handles basic arithmetic, algebra, derivatives, and integrals.
- **LaTeX Rendering**: Beautiful mathematical notation using MathJax.
